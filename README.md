# Claude Code Academy

A hands-on practice repository for the [Anthropic Academy - Claude Code Course](https://docs.anthropic.com/en/docs/claude-code).

## Structure

| Folder | Purpose | Module |
|--------|---------|--------|
| `.claude/commands/` | Custom slash commands | 6 |
| `.github/workflows/` | GitHub Actions integration | 8 |
| `hooks/` | Pre/post tool-use hooks | 9-12 |
| `scripts/` | SDK experiments | 13 |
| `src/` | Practice source code | Various |

## Quick Start

cd ~/claude-code-academy
claude
